Portfolio

HeartBeat

Project realized for emotions visualization, based on the analysis of text which we import to the software.

CA Advisor Support App

Advisor support application for Android and iOS has a convenient and straightforward interface which allows users to use the following instruments with ease: tax calculator, loan calculator, reminders of tax returns and payments.

The World Nomad Games website

In September 2014, on the shore of Lake Issyk-Kul, in Cholpon-Ata, the First World Games of Nomads were held. And before our team was the task of creating a bright and impressive wow-site about main events of this games.

Garder

The system is intended for inspectors of the city water utility. It allows to inspect various objects for water quality, record violations, control the sending of notices to violators, receive reports on previously detected violations. With the help of this system, it became much more accessible to monitor the current state of the equipment.

Elcard Mobile

The Interbank Processing Center (IPC) was intended to enter the market of individuals to provide online payment services. Client independently developed a back-end for the system. Attractor Software was responsible for the development of the client’s part of the mobile application on Android platform.

Digital student book

The purpose of the project is to create the base for piloting of first digital books in Kyrgyz education system. Existing digital books don't allow users to create colourful illustrations or structure text according to student book’s table of content.

Stom.Kg

CRM-system, which helps dentists work with their patients. The application was optimized for mobiles and tablets. It simplified the process of day-to-day dentists’ work that allows keeping records of patient records and doctors, track applications, payments and expenses.

Tengri Wallet

This project was built as the foundation for the possibility of connecting several banking systems

The map of legal service providers

The interactive online map is an instrument that allows to search for legal service providers in the certain region. The map contains a broad range of providers across Kyrgyzstan.

UniForm

System for the Canadian Government which helps for data collection, reporting tool and visualization of them. This instrument will assist the government in capturing valid, reliable, complete, and timely data related to the performance of their social investments.
